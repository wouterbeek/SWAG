# Sackner Archives

This is the transformation script for the Linked Data version of the
Sackner Archive.

## Running the script

```sh
$ swipl -s script.pl -g run -t halt
```

## Online publication

THe Linked Data version of the Sackner Archive is published
[online](https://triplydb.com/wouter/sa).
